A note about forks

On August 1, 2017, multiple development teams forked from the legacy bitcoin chain to remove the 1 megabyte block size restriction, creating Bitcoin Cash (BCH). The impact of this change is visible on several charts.

On August 24, 2017, the Bitcoin Core development team forked from the legacy bitcoin chain to implement a feature called SegWit, creating Bitcoin Core (BTC). Data for this product can be found at Bitcoin Core Charts.
